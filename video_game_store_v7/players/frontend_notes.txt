axios: library to send http requests to node.js

react-router useNavigate: allows navigation from one component to the other using routes

React.FormEvent: events that ocurr during form interatcions e.g submit a form or modify input field

React Router <Link>: Substitute for href, useful for buttons. 

Hooks: Allow use of built in React Features.
 * useState - A Hook that lets you add state to functional components.
              React re-renders the component when the state changes. 
 * useEffect - A Hook that lets you perform side effects in function components. 
            These include:
               Fetching data, Subscribing to events, Manually modifying the DOM and
               Setting up timers. 
               It runs after render 

Props: Data that can be shared between components.
